1. Add the three .jar files in the /libraries folder to your classpath through eclipse by clicking:
	Project->Properties->Java Build Path->Libraries->Add External Jars...

2. Add the log4j configuratio folder (/libraries/log4j2_config) to your classpath through eclipse by clicking:
	Project->Properties->Java Build Path->Sources->Add Folder...

3. Change server, nick, login, and channel strings to desired values in the HackBot.java file.

4. Project should be ready to run.
